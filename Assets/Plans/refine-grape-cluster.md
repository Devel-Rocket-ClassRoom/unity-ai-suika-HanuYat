# 프로젝트 개요
- 게임 제목: 수박 게임 클론 (Unity AI Suika)
- 수정 목표: 포도 스프라이트를 단일 공 모양에서 '포도송이(Cluster)' 형태로 변경. 기존의 귀여운 카툰 스타일과 얼굴은 유지.

# 구현 단계
## 1. 에셋 생성 (Refinement)
- 1.1. `Unity.AssetGeneration.GetModels`를 사용하여 모델 확인.
- 1.2. 포도 스프라이트 재생성. '포도송이(Cluster of grapes)' 형태이며, 기존 포도의 색감과 귀여운 얼굴을 유지하도록 프롬프트 구성. `referenceImageInstanceId`를 사용하여 스타일 일관성 유지.
- 1.3. 생성된 스프라이트의 배경 제거 (`RemoveSpriteBackground`).

## 2. 프로젝트 통합
- 2.1. 기존 `Grape.png`를 새로 생성된 파일로 교체.
- 2.2. `FruitData` 및 프리팹 참조 확인.
- 2.3. 스케일 및 콜라이더 설정 재검토. 포도송이 형태에 맞춰 `CircleCollider2D`가 적절한 크기로 설정되도록 조정.

# 검증 및 테스트
- 시각적 확인: 에디터에서 포도가 송이 형태로 잘 바뀌었는지 확인.
- 게임 플레이: 인게임 병합 및 물리 동작 확인.
